# Java_P01 — Roguelike Game

Консольная игра в жанре roguelike на Java с процедурной генерацией уровней.

## Технологии
- Java
- Gradle / Maven
- Работа с файлами (JSON)

## Структура

src/roguelike/
├── domain/ # Логика игры (генерация, бой, предметы)
├── data/ # Репозиторий для сохранения/загрузки
├── presentation/ # Консольный UI
└── util/ # Вспомогательные классы


## Особенности
- Процедурная генерация подземелья
- Боевая система (игрок vs монстры)
- Инвентарь и предметы
- Сохранение / загрузка состояния (JSON)
- Туман войны

## Запуск

cd src/roguelike
./gradlew run


# или
  ./mvnw compile exec:java
