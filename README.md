# Java Bootcamp Projects

Коллекция проектов с интенсива по Java (School 21 / Ecole 42). Проекты разделены на несколько модулей, каждый из которых охватывает определённые темы: от основ языка до многопоточности и работы с базами данных.

## 📚 Структура репозитория

- **`Java_P01`** — Roguelike игра на Java (Maven/Gradle). Отработка ООП, работа с файлами, генерация уровней.
- **`Java_T01`** — Набор небольших задач на Java (алгоритмы, работа с коллекциями, ввод/вывод).
- **`Java_T02`** — Темы: наследование, полиморфизм, интерфейсы, абстрактные классы, итераторы.
- **`Java_T03`** — Веб-приложение "Крестики-нолики" (Spring Boot, REST API, трёхуровневая архитектура).

## 🛠 Технологии и инструменты

- **Java** (основной язык)
- **C** (некоторые вспомогательные модули)
- **Makefile** (сборка)
- **Maven / Gradle** (управление зависимостями)
- **Spring Boot** (только `Java_T03`)

## 📖 Содержание проектов

### `Java_P01` (Roguelike)
- Консольная игра с процедурной генерацией уровней.
- Реализация боевой системы, инвентаря и сохранения состояния.
- Паттерны проектирования (Repository, Service, Presentation).

### `Java_T01` (Алгоритмы и основы)
- Решение классических задач: поиск, сортировка, работа со строками и файлами.
- Знакомство с коллекциями, исключениями, вводом/выводом.

### `Java_T02` (ООП)
- Наследование, полиморфизм, инкапсуляция.
- Создание иерархий классов (животные, транспорт и т.д.).
- Реализация итераторов для кастомных коллекций.

### `Java_T03` (Веб-приложение "Крестики-нолики")
- REST API на Spring Boot.
- Трёхуровневая архитектура (контроллер → сервис → репозиторий).
- Хранение данных в памяти (с возможностью расширения до БД).
- Обработка ошибок и валидация.

## 🚀 Запуск

Каждый проект является самостоятельным и запускается отдельно.

### Для проектов на Java (Maven/Gradle):
```bash
# Перейдите в папку проекта, например:
cd Java_T03/src/tictac

# Для Gradle:
./gradlew bootRun

# Для Maven:
./mvnw spring-boot:run
