# Java_T01 — Алгоритмы и основы Java

Набор небольших задач на Java для отработки базовых навыков.

## Темы
- Алгоритмы поиска и сортировки
- Работа со строками и числами
- Чтение/запись файлов
- Коллекции (List, Map, Set)
- Обработка исключений

## Упражнения
| № | Название | Описание |
|---|----------|----------|
| 0 | TriangleMain | Работа с треугольниками |
| 1 | TimeSearch | Поиск времени |
| 2 | FibonacciModule | Модуль Фибоначчи |
| 3 | NegativeAverage | Среднее отрицательных |
| 4 | FirstLastDigit | Первая и последняя цифра |
| 5 | SelectionSort | Сортировка выбором |
| 6 | FileReader | Чтение файлов |
| 7 | LineNumber | Нумерация строк |
| 8 | StringScanner | Сканер строк |
| 9 | User/Main | Работа с пользователями |

## Запуск
Каждое упражнение — отдельный Gradle-модуль:

cd src/exercise1
./gradlew run
