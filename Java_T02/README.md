# Java_T02 — Объектно-ориентированное программирование

Отработка принципов ООП на Java: наследование, полиморфизм, инкапсуляция, абстракция.

## Темы
- Наследование и переопределение методов
- Полиморфизм (раннее и позднее связывание)
- Абстрактные классы и интерфейсы
- Итераторы для кастомных коллекций

## Упражнения
| № | Описание |
|---|----------|
| 1 | Базовые классы Animal, Cat, Dog |
| 2 | Расширение иерархии |
| 3 | Добавление Herbivore, Omnivore, GuineaPig, Hamster |
| 4 | Работа с классами и объектами |
| 5 | Демонстрация полиморфизма |
| 6 | Реализация кастомного итератора AnimalIterator |

cd src/exercise1
./gradlew run
